# JARVIS OS Vault

A personal Obsidian-style knowledge vault covering JARVIS OS architecture, AI models and prompt engineering, automation, memory systems, working principles ("Metodologia Gabriel"), video engineering, and photography. Notes are flat under `03-Notes/`, organized by linking rather than folders, and surfaced through domain hub notes in `02-Hubs/`. This vault also doubles as the live memory source for the JARVIS OS app's 3D Vault Brain — see `CLAUDE.md` for the structural rules that keep that rendering correct.
