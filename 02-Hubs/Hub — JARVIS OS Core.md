## Definition

JARVIS OS is a voice-first AI assistant built on React, Vite, and Three.js, whose long-term memory is this vault. Its [[Voice-First Interface]] is the primary way it's used; its 3D [[Vault Brain Concept]], built with [[Three.js Visualization]], is a supporting visualization of that memory, not a replacement for it.

## Contained clusters

The Vault Brain is where this vision becomes concrete: it renders every note in this vault as a node and every wikilink as an edge, live, via the File System Access API. That rendering is what makes [[Bridge — Vault as Mind]] literal rather than metaphorical — this vault's [[Graph-Based Memory]] is JARVIS OS's memory.
