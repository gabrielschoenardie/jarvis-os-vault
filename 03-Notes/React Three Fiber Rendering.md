## Concept

React Three Fiber is the React renderer for Three.js used to build jarvis-os's 3D Vault Brain — it lets the graph of notes be expressed as declarative React components (nodes, edges, camera) instead of imperative Three.js calls.

## Connects to

Part of [[Hub — Architecture]], applying [[Component-Based UI]] to a 3D scene graph specifically for [[Three.js Visualization]], and it is the rendering layer underneath [[Bridge — Vault as Mind]].
