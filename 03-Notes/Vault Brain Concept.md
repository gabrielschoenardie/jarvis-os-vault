## Concept

The Vault Brain is jarvis-os's 3D rendering of this vault's note graph, read directly from the vault's Markdown files via the File System Access API — turning [[Graph-Based Memory]] into a navigable visual space rather than a folder tree.

## Connects to

Part of [[Hub — JARVIS OS Core]], rendered with [[Three.js Visualization]] and [[React Three Fiber Rendering]], and the direct subject of [[Bridge — Vault as Mind]].
